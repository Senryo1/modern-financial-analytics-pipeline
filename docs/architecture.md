
# Project Architecture

## Initial Data Workflow

Raw Data → Python Cleaning → SQL Validation → Analytics Models → Power BI Dashboard → Documentation

## Planned Architecture Layers

### 1. Raw Layer

Stores original source data without transformations.

### 2. Processed Layer

Stores cleaned and standardized datasets generated by Python scripts.

### 3. Analytics Layer

Stores business-ready tables and queries for reporting and analysis.

### 4. Dashboard Layer

Presents executive insights through Power BI.

## Future Improvements

- Add dbt models.
- Add Snowflake or cloud warehouse layer.
- Add automated data quality tests.
- Add orchestration with Airflow.
